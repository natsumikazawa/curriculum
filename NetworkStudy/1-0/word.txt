■ネットワーク
　・LAN ... Local Area Networkの略であり、家庭内や社内などの限定的なネットワーク。
	
	・有線LAN ... ケーブルを使ったネットワーク。セキュリティが高い。
	・無線LAN ... 電波で接続されたネットワーク（Wifiなど）。

　・WAN ... Wide Area Networkの略であり、遠く離れた地点を繋ぐネットワーク。
	　　LAN同士を繋いで、広範囲のネットワークを形成したものを表す。
	　　ただしインターネットとは違い、特定の人が接続できるネットワーク（プライベートネットワーク）である。

	・インターネット ... 世界中で繋いでいるネットワーク。（オープンネットワーク）

■ネットワーク構成
　・ノード ... パソコンやネットワーク接続のあるプリンタなどの、ネットワークに繋がっている１つ１つの機器のこと。
　・リンク ... ノードとノードをつなげているものをリンクと言う。（ケーブルなど）
　・トポロジ ... コンピュータネットワークの接続形態のことで、ノードがどのような形で接続するのかを表す。

	・バス型トポロジ ... １本のケーブル（同軸ケーブル）に複数のノードを接続しているトポロジ。同軸ケーブルに障害が発生すると、それに接続している全てのノードが通信できなくなる。
	・スター型トポロジ ... １つの集線装置に複数のノードを接続しているトポロジ。１本のケーブルに障害が発生しても、そのケーブルに繋いでいるノード以外には支障をきたさない。ただし、集線装置に障害が発生すると、全てのノードが通信できなくなる。
	・メッシュ型トポロジ ... 全てのルーターと相互接続しており、特定のルーターとのリンクに障害が発生しても、他のルーターとのリンクを介して継続して通信することができ、耐障害性に優れている。

　・帯域 ... ネットワークの世界での「道幅」のようなもの。
	
	・bps ...　１秒間にどれだけ多くのビットを送信できるかを表す。例えば40bpsだと、１秒間に40ビット送信する、という事になる。帯域をbpsで表される。

　・輻輳 ... 帯域が狭いと、処理できずにフレームを捨ててしまう。これを輻輳と言う。

　・コリジョン ... 無線LANなどで複数の端末が送信し、データが衝突する現象のこと。
　・ドメイン ... インターネット上での「住所」のようなもの。（xxx@yyy.comのyyy.comの部分）
　・ポート ... インターネット上での「扉」のようなもの。
	　　　サーバ側のポートは固定番号でいつも開いている状態で、PC(クライアント）側のポートはセキュリティの都合上、不定番号で開いたら閉じる、を繰り返す。

■モデル
　・プロトコル ... 「通信する上での約束事」を意味する。
　・OSI参照モデル ... ネットワークを理解する上でのプロトコル。違うメーカーの機器同士でも通信ができるようにするために、「Open System Internet（OSI）」の設計方針に基づいて通信機能を分割して階層化したのが0SI参照モデルである。７つの階層から構成される。
　・ヘッダ ... データを送受信する上での、データの荷札のようなもの。データをどのように処理するのか、などの情報を記して、データと一緒に送受信する。
　・カプセル化 ... 上位の層から下位の層にデータを受け渡すときに、データと一緒にヘッダを付けて送ることをカプセル化という。
　・非カプセル化 ... 下位の層から上位の層にデータを送るときに、ヘッダを取り除きながらデータを送っていくことを非カプセル化という。
　・PDU ... 「プロトコルデータユニット」の略。あるプロトコルが扱うひとまとまりのデータの送受信単位のこと。（例：TCP → パケット、イーサネット → フレーム、など）
　	・MTU ... 「マキシマムトランスミッションユニット」の略。ネットワーク上で一度の通信で転送可能なデータのサイズ制限のこと。
　	・フレーム ... イーサネットなどのL2層の通信で使われるPDUの呼び名。
　	・パケット ... IPなどのL３層の通信で使われるPDUの呼び名。
　	・セグメント ... TCPなどのL４層の通信で使われるPDUの呼び名。

　・TCP/IPモデル ... OSI参照モデルだと実装の際に複雑すぎたりと様々な問題がある。そこで、シンプルで実用的な階層構造にしたものがTCP/IPモデルである。４つの階層から構成される。

■物理層
　・ビット ... コンピュータが処理するために用いられるデータの桁数をビットという。
　・ツイストペアケーブル ... 芯線を２本対でツイストして１ペアにした形状のケーブルを指す。ノイズを受けにくくすることができる。
	・UTP ... アンシールデッドツイストペアの略。シールド無し、という意味。
	・STP ... シールデッドツイスとペアの略。芯線をアルミシールドでカバーすることにより、電磁波やノイズに強い耐性を持ったケーブル。
	・カテゴリー ... LANケーブルは、通信速度などでカテゴリーが分けられている。カテゴリーの数字が大きいと、その数字以下のカテゴリーの全てに対応可能。

　・コネクタ ... ケーブルの先端にある、LANポート（受け口）に接続するための端子。
　・ストレートケーブル ... LANケーブルの種類。コネクタの中の細い線の並びが左端と右端で同じもののことを指す。他の機器同士（コンピュータとスイッチ、コンピュータとルーター、など）を繋ぐときに使われる。
　・クロスケーブル ... LANケーブルの種類。コネクタの中の細い線の並びが左端と右端で違うもののことを指す。基本的にパソコン同士を繋ぐときに使われる。
　・光ファイバケーブル ... 光ファイバ心線などを束ねて、屋内外での使用に耐える構造にしたものが光ファイバケーブルである。電気信号ではなく半導体レーザーやLED等の光によって通信している。長距離で通信することが可能。
　・ハブ ... 有線LANを構成するための集線装置、つまり、ネットワークのケーブルが集まっている装置のこと。
	・半二重通信 ... LANの通信方式。一度に「送信」と「受信」のどちらかしか行うことができない。

■データリンク層
　・イーサネット ... パソコンなどの機器を優先接続する際の通信規格（プロトコル）の１つ。イーサネットが登場したことで、メーカーの異なる機器同士でも接続できるようになった。
	・イーサネット ... 通信速度が１０Mbpsのもの。同軸ケーブルなどが利用可能。
	・ファスト・イーサネット ...  通信速度が１００Mbpsのもの。光ファイバーケーブルなどが利用可能。
	・ギガビット・イーサネット ... 通信速度が１Gbpsのもの。現在では最も広く使用されている。

　・イーサネットヘッダ ... 簡潔に説明すると、「イーサネット（で使う）ヘッダ（情報）」。通信する際にデータにくっつけられた色々のうち、イーサネットで使う情報がイーサネットヘッダである。
　・トレーラ ... 相手のコンピュータが受信したときに、伝送途中に壊れていないか確認するためのチェックコードのことである。データリンク層でつけられることが多い。
　・CSMA/CD ... LANネットワークで用いられる媒体アクセス制御（MAC)方式のひとつで、通信路の使用状況を監視して、伝送路の空きを見つけてデータ伝送を行う方式のことである。
	　　　　複数の端末が同時にデータを発信すると、コリジョンが起きてしまう。そのために、CSMA/CDでは、あらかじめ伝送路上に他のキャリアがあるかどうか確認し、存在していることを確認した際には両者とも送信を中止して、一旦待機する。
　　　　　　　　　ランダムな時間を待機し、その後で送信を再開する。そうすると、単一の伝送路でも相互に通信経路を用いることが可能である。
　　　　　　　　　ただし、トラフィックが一気に増大すると待ち時間が膨れ上がってしまう、という難点もある。
　・MACアドレス ... Media Access Control アドレスの略。パソコンやルーターなどのネットワーク機器についている番号のことで、イーサネット通信で利用されている。
　　　　　　　　　　　通信データのヘッダ部分に「送信元アドレス」と「送信先アドレス」を格納することで、宛先や戻し先を確定している。
　・スイッチ ... 集線装置の一つ。従来のリピーターハブにMACアドレスを用いた、宛先によるデータ分配機能を追加した装置である。

　	・MACアドレステーブル ... ネットワークスイッチなどが内部的に管理している情報で、どのポートにどのMACアドレスを持つ機器が接続されているかを記録しているもの。
	・フラッディング ... スイッチでは、あるポートから受信したフレームの宛先が分からず、全ポートに転送してしまうことをフラッディングという。
	・全二重通信 ... データの「送信」と「受信」を同時に行えることを全二重通信という。

　・VLAN ... Virtual Local Area Networkの略。物理的な接続形態とは別に、論理的なLANセグメントを作る技術。
　　　　　　　　スイッチなどによって、どのコンピュータとどのコンピュータがやりとりできる、あるいはできないかを制御することで、擬似的にネットワークを分割したり結合したりする仕組み。
	・タグVLAN ... VLANのやり方の１つ。データにそのデータが所属するネットワークの情報を付与し、制御する側はその情報を見て所属するネットワークを判断するやり方。
　　　　　　　	　　　例）名札をデータにくっつけて、その名札を元に制御するようなイメージ。
	・ポートベースVLAN ... VLANのやり方の１つ。スイッチのどの穴に線を挿すかで、どのネットワークに所属するかが決まる。
		・アクセスポート ... １つのネットワークを形成する穴のこと。主にPCやプリンターなどの機器とスイッチを接続するために使われる。
		・トランクポート ... 複数のネットワークを形成する穴のこと。スイッチ同士をつなげるように設定することで、他のスイッチとネットワークを形成できる。
	・DTP ... Dynamic Trunking Protocol の略。スイッチポートをアクセスポートまたはトランクポートのどちらにするのかを動的に決定するシスコ社独自のプロトコル。
		　DTPによって、対向のスイッチポートの設定状況に応じて自身のスイッチポートをアクセスポートまたはトランクポートで動作する。
	・デフォルトVLAN ... 設定が未設定の状態でも、装置の起動後すぐにL２中継ができる。このとき、全てのスイッチポートはデフォルトVLANと呼ぶVLANに属す。デフォルトVLANは常に存在し、VLAN ID には「１」を使う。
	・ネイティブVLAN ... トランクリンク上で唯一タグを付けずに送受信されるVLANのこと。デフォルトではVLAN IDが「１」のネットワークがネイティブVLANとなっている。
			　VLANを設定できないコンピュータなどを接続したり、経路上にVLANを理解しない中継機器がある場合でも通信ができるようになる。
　・ブロードキャストストーム ... ネットワークにおいて、一斉送信用の特殊データや信号が転送され続ける現象。２台以上のネットワークでループが形成されることで発生し、最終的にネットワークの全体機能が停止する。
　・スパニングツリープロトコル（略：STP）... ループ状に形成されたL２ネットワークにおいて、データトラフィックが永続的に流れ続けることを防止するプロトコル。
				　　ブロードキャストストームなどの状態が起きないように、STPを利用する。そうすると、あるポートが自動的にブロッキング状態になるが、障害発生時には自動的にそのポートが有効化される。
　
　・リンクアグリゲーション（略：LAG） ... 複数の回線を束ねて一体的に運用し、仮想的に１本の回線と見なすことで、通信速度及び耐障害性を向上させる技術。
　・EtherChannnel ... シスコ社での「リンクアグリゲーション」の呼び名。
