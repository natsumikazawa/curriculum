〈ネットワーク層〉
・IP ... Internet Protocolの略。相手先のアドレス情報をもとにしてパケットを送信する機能
・IPヘッダ ... IPでつけられるヘッダ。送信元IPアドレスや宛先IPアドレスが記載されている。
・ICMP ... Internet Control Message Protocolの略。IP通信の疎通状態を調査したり、正常に通信できない場合には送信元ホストへエラー情報を返したりする、IP通信を補佐するためのプロトコル。
・ICMPメッセージ ... エラーの通知や通信状態の確認を実行するもの。ErrorメッセージとQueryメッセージの２種類がある。
・IPアドレス ... ネットワークに接続されたパソコンやプリンターなどに割り当てられる番号。ネットワーク通信を行う場合に利用される。
	・オクテット ... IPv4アドレスにおいて、８ビットずつカンマで区切られた部分
	・ネットワーク部 ... ネットワークを識別するアドレス。基本的に変更はできない。
	・ホスト部 ... コンピュータや各機器を識別するアドレス。個々の設定で変更できる。
	・クラス ... IPv4にはクラスというものが存在し、それぞれ「クラスA」「クラスB」「クラスC」に分けられている。
	・プライベートIPアドレス ... LANなどの企業内ネットワークに存在する端末が、自由に使用できるIPアドレスのこと。
	・グローバルIPアドレス ... インターネットを利用するときに割り振られるIPアドレスのこと。
・ネットワークアドレス ... あるネットワークに割り当てられたIPアドレス群の中で、そのネットワーク自身を指し示すアドレス。また、アドレスを表すビット列のうち、ネットワークを表す上位ビット部分のみを指す場合もある。
・ブロードキャストアドレス ... ネットワーク内の全ての機器にデータを一斉送信するために使われる特殊なアドレス。
・ループバックアドレス ... そのコンピュータ自身を示すIPアドレスのこと。あるコンピュータがネットワークを通じて提供している機能に自信で動作する別のソフトウェアからアクセスする場合や、ネットワークを利用するソフトウェアのテストなどに用いられる。
・サブネット ... 大きなネットワークの中にある小さなネットワーク。
・サブネットマスク ... IPアドレスを分割して、どこがネットワーク部で、どこがホスト部かを識別するために使う数値。
	・プレフィックス表記 ... IPアドレスとサブネットマスクを同時に表現する書き方。
・デフォルトゲートウェイ ... 内部ネットワークと外部ネットワークを接続するためのノード。
・IPv6 ... IPv4に次ぐインターネットプロトコルの規格。約３４０澗個IPアドレスが用意できるようになる。
	・サブネットプレフィックス ... IPv6を構成するものの１つ。サブネットプレフィックスの長さが、IPv4のサブネットマスクに相当する。
	・インターフェイスID ... IPv6を構成するものの１つ。特定のノードのインターフェースを識別する。
・ルータ ... コンピュータネットワークにおいて、データを２つ以上の異なるネットワーク間に中継する通信機器。
	・ルーティングテーブル ... ルータに記録される経路情報で、ルーティング処理を行う際に参照する。
		・ネクストホップ ... ルーティングテーブルに記載される情報の１つで、個々の宛先ネットワーク毎に決められた、次に転送すべき隣接ルータ（のIPアドレス）のこと。
	・ARPテーブル ... Address Resolution Protocolテーブルの略。ARPによって割り出したIPアドレスとMACアドレスの対応表。
		・ARP ... Address Resolution Protocolの略。与えられたインターネット層アドレスに対応するデータリンク層アドレスを発見するために使用される通信プロトコル。
	・コアルータ ... 基幹ネットワークのバックボーンを構成するルータで、高速なパケット転送を実現する。大型のルータが適用される。
	・センタールータ ... 企業の拠点間WANネットワークで本社拠点やデータセンターなどの、企業で中心的に利用されるネットワークを接続するために利用される。
	・エッジルータ ... 企業の拠点間WANネットワークで支社や営業所などの、拠点ネットワークを接続するために利用される。
	・ブロードバンドルータ ... 家庭環境やSOHO環境でインターネット回線に接続する目的で利用される。
・ルーティング ... 異なるネットワークにパケットを送信するときに最適な配達経路を決めること。
	・直接接続 ... ルーティングテーブルにルート情報を登録する方法の１つで、最も基本的なルート情報の登録方法。ルータが直接接続しているネットワークのルート情報。
	・スタティックルーティング ... ルータなどが管理者の入力、設定した経路表（ルーティングテーブル）に従って固定的に経路選択を行うこと。静的ルーティングとも呼ばれる。
	・ダイナミックルーティング ... ルータなどが経路情報を互いに交換しあい、自動的に生成、更新し続けるルーティングテーブルに基づいて経路選択を行うこと。動的ルーティングとも呼ばれる。
		・ディスタンスベクタ型 ... ルーティングプロトコルの特徴、分類の１つで、隣接するルータ間でルーティングテーブルを交換しあい、どの隣接ルータを経由すれば最短のホップ数で宛先に届くかを基準に経路を指定する方式。
		・リンクステート型 ... ルーティングプロトコルの特徴、分類の１つで、どのルータとどのルータが隣接しているか、という接続情報（リンクステート）を交換しあい、この情報の集合に基づいて経路を選択する方式。
		・ハイブリッド型 ... ルーティングプロトコルの特徴、分類の１つで、ディスタンス型を基本としつつ、リンクステート型の特徴を兼ね備えたルーティングプロトコル。


<トランスポート層>
・コネクション型通信 ... データ通信において、通信を開始する前に相手との間で仮想的な専用通信路（コネクション）を確立し、それを通じてデータの送受信を行う通信方式。送信したデータが相手に確実に届き、データの到着順も送信順通りになることが保証される。
	・ACK ... ACKnowledgementの略。データ通信において、受信側から送信側に送られる、データを正常に受信したことを示す信号。ACKが届かない場合には送信側は同じデータを再送する。
・コネクションレス型通信 ... 通信相手の状況を確認せずにデータを一方的に送りつける方式。送信したデータが確実に相手に届くかどうかや、送信したデータの到着順は保証されない。
・TCP ... Transmission Control Protocolの略で、IPと同様にインターネットにおいて標準的に利用されているプロトコルである。ネットワーク層のIPとセッション層以上のプロトコルの橋渡しをする形で動作している。
	・スリーウェイハンドシェイク ... IPネットワーク上でTCPによる接続（コネクション）を確立するための手順。パケットの送受信を3回行うことからこのように呼ばれる。
	・Telnet ... ネットワークに接続された機器を遠隔操作するために使用するアプリケーション層プロトコル。
	・HTTP ... Hypertext Transfer Protocolの略。WebサーバとWebブラウザの間で、Web情報をやりとりするためのプロトコル。
	・FTP ... File Transfer Protocolの略。サーバとクライアントの間でファイル転送を行う際に必要となる通信プロトコル。
	・SMTP ... Simple Mail Transfer Protocolの略。メールを送信するための通信プロトコル。
・TCPヘッダ ... TCPセグメントの先頭にある制御情報が記された部分のこと。
・UDP ... User Datagram Protocolの略。インターネットなどで使用される通信プロトコルであり、トランスポート層のプロトコル。
	・RTP ... Real-time Transfer Protocolの略。音声や動画などのデータストリームをリアルタイムに配送するための通信プロトコル。
	・RTCP ... Real Time Control Protocolの略で、RTPとセットで使われる。RTPのサービス品質についてフィードバックを提供する。
	・DHCP ... Dynamic Host Configuration Protocolの略。IPv4ネットワークで使用されるネットワーク管理プロトコルであり、コンピュータがネットワークに接続する際に必要な設定情報を自動的に割り当てるために使用する。
	・TFTP ... Trivial File Transfer Protocolの略で、UDP上で動作するファイル転送プロトコルである。送受信したいファイル名をtftpサーバに送ると、そのファイルの内容をUDPで１パケットずつに分割して送り返してくる。
	・RIP ... Routing Information Protocolの略。ルータなどの通信機器の間で経路情報を交換し、ある地点から別の地点までの最短経路を割り出すのに使われる。
・UDPヘッダ ... UDPセグメントの先頭にある制御情報が記された部分のこと。送信元ポート、宛先ポート、データ長、チェックサムで構成されている。
